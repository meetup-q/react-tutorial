# star-wars-app
我們現在應該很熟悉使用
首先利用 `npx` 創建一個新的 react 專案，`npx` 是 `npm` 的指令，這個指令會檢查當前路徑中的是否存在 `create-react-app` 工具，若無則會先下載 `create-react-app` 工具後再刪除。

```shell
npx create-create-app star-wars-app
```
